Uses the custom header as the login logo.

== Description ==

This plugin replaces your login logo with your custom header. No configuration needed, just install, activate and you are good to go!
Supports Wordpress Multisite/Network.


== Installation ==

This is how you install and use the plugin.

1. Upload the folder `header-login-logo` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Sit back and enjoy your new login logo

This plugin has no configuration page because there is no need to configure anything. Just make sure your theme supports [Custom Headers](http://codex.wordpress.org/Custom_Headers). The plugin will resize and make sure your new logo looks good.
